VARIABLES: Las variables son palabrasn o caracteres a las que les puedo asignar un valor, podria decirse que es un contenedor que guarda el valor que le asigne y que despues puedo recurrir a ella para utilizarla, ejemplo: luego de que a la palabra "hola" la tranformo en variable y le asigno un valor "12 dias", la palabra "hola" valdra 12 dias.

STRINGS: Los strings son cadenas de caracteres encerrados entre comillas. 

FUNCIONES: Las funciones son un grupo de codigo al que le puedo asignar una tarea y luego utilizarla o invocarla cuando la necesite. 

  ARGUMENTOS: Los argumentos son como unas variables que se utilizan dentro de las funciones.
 
  RETURN: El return se utiliza para que la funcion te debuelva un valor

DECLARACIONES IF: El if se utiliza para ejecutar diferentes codigos segun se cumplan las condiciones 

VALORES BOOLEANOS TRUE, FALSE: se utilizan para ejecutar un bloque de codigo si es true(verdadero) y otro si es falce (falso)
